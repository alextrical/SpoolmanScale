SpoolmanScale
Filament Scale with NFC & Spoolman Integration

Before Flashing

  1. Connect USB cable to PC
  2. Disconnect debug board (ZXACC-ESPDB)
  3. Click "Install" below
  4. Select port & confirm
  5. After flashing: set up WiFi via the Welcome Screen
⚠ Note on First Flash The first flash will erase all saved settings (WiFi, calibration). These can be re-entered on the device afterwards.
